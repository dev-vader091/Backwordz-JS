The Code

          
            function getValues() {
              // get the value of the input and create variable to be used
              // as parameter in logic function
              let inputString = document.getElementById('userString').value;
          
              // create a variable to hold value returned in logic function 
              let reversedString = reverseAString(inputString);
              
              // after running logic function -- the returned value is used as
              // the paremeter in the display function
              displayString(reversedString);
          }
          
        
          
            function reverseAString(userString) {
              // initialize a variable to represent the string reversed
              let revString = "";
              // loop through parameter value -- starting from last index
              for(i = userString.length - 1; i >= 0; i--) {
                  // create a variable to repesent the current character i
                  let currentLetter = userString[i];
                  // add the current i value to our initialized variable 
                  revString += currentLetter;
              }
              // return the new string
              return revString ;
          }
          
        
          
            function displayString(reversedString) {
              // get the element that will hold the value of the new string from the document
              // and make the string the text 
              document.getElementById('results').textContent = reversedString;
              // remove the invisible class from the parent element so the user can see new string
              document.getElementById('alert').classList.remove('invisible');
          }
getValues()

This function is the control function and entry point of our program. To start, we get the value of the input and create a variable, inputString, to be used as a parameter in the logic function. Next a variable, reversedString, is initialized to hold the value returned in the reverseAString() logic function. After running the reverseAString() function, the returned value is used as the parameter in the view function displayString() .

reverseAString(userString)

This is the logic function where we first initialize a variable, revString, as an empty string to represent the string provided as the parameter userString reversed. Next, looping through the userString value from the last index, create a variable, currentLetter, to repesent the current character i. At the end of each iteration, we add the currentLetter value to the revString variable. After the loop, we return the revString value.

displayString(reversedString)

This function is the view function that displays results to the user on the browser. First we get the element that will hold the value of the new string from the document and make the reversedString parameter value the displayed text with the textContent property. Last, remove the invisible class from the parent element so the user can view new string.
